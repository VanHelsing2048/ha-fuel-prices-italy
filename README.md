# Fuel Prices Italy

Home Assistant custom integration for daily fuel station prices in Italy, powered by the public MIMIT Osservaprezzi Carburanti web service.

> This integration is for Italy only.

Repository: [VanHelsing2048/ha-fuel-prices-italy](https://github.com/VanHelsing2048/ha-fuel-prices-italy)

## Features

- Search fuel stations around a latitude/longitude point.
- Configure the search radius in kilometers.
- Select which fuel types to monitor.
- Choose self-service prices, serviced prices, or both.
- Create Home Assistant sensors for the best nearby prices.
- Create `geo_location` entities so stations can appear as markers on Home Assistant maps.
- Configure everything from the Home Assistant UI.
- HACS-ready repository layout with changelog support.

## Installation With HACS

1. Open HACS in Home Assistant.
2. Go to **Integrations**.
3. Open the three-dot menu and choose **Custom repositories**.
4. Add this repository URL: `https://github.com/VanHelsing2048/ha-fuel-prices-italy`.
5. Select category **Integration**.
6. Install **Fuel Prices Italy**.
7. Restart Home Assistant.
8. Go to **Settings > Devices & services > Add integration**.
9. Search for **Fuel Prices Italy**.

## Manual Installation

Copy this directory:

```text
custom_components/fuel_prices_italy
```

into your Home Assistant configuration directory:

```text
config/custom_components/fuel_prices_italy
```

Then restart Home Assistant and add the integration from the UI.

## Configuration

The integration asks for:

- **Latitude**: center of the search area.
- **Longitude**: center of the search area.
- **Search radius**: radius in kilometers.
- **Fuel types**: fuels to monitor.
- **Show self-service prices**: include self-service fuel prices.
- **Show serviced prices**: include served fuel prices.
- **Update interval**: polling interval in minutes.
- **Maximum stations**: maximum number of station marker entities.

Default settings:

- Radius: `5 km`
- Fuel types: Benzina, Gasolio
- Mode: self-service only
- Update interval: `60 minutes`
- Maximum stations: `50`

## Created Entities

### Sensors

The integration creates best-price sensors such as:

- `sensor.fuel_prices_italy_benzina_self_migliore`
- `sensor.fuel_prices_italy_gasolio_self_migliore`
- `sensor.fuel_prices_italy_gpl_migliore`
- `sensor.fuel_prices_italy_distributori_trovati`
- `sensor.fuel_prices_italy_distributore_piu_vicino`

Best-price sensors expose useful attributes:

- station id
- station name
- brand
- distance in km
- last price update from MIMIT
- fuel id
- fuel name
- self-service flag

### Map Markers

Each station is exposed as a `geo_location` entity with:

- latitude
- longitude
- distance
- brand
- visible fuel prices
- MIMIT update timestamp

These entities can appear in Home Assistant map views and Map cards.

Example Map card:

```yaml
type: map
entities:
  - entity: geo_location.x_oil
  - entity: geo_location.distributore_carburanti_tap
hours_to_show: 24
```

Entity ids are generated by Home Assistant and may differ from the example.

## Data Source

Fuel Prices Italy queries the MIMIT Osservaprezzi Carburanti endpoint used by the public Italian fuel-price search application:

```text
https://carburanti.mise.gov.it/ospzApi/search/zone
```

The integration sends:

```json
{
  "points": [{"lat": 45.644912, "lng": 12.330887}],
  "fuelType": 1,
  "priceOrder": "asc",
  "radius": 5
}
```

The remote API returns nearby stations, coordinates, prices, service mode and update timestamps.

## Update Policy

Use a reasonable polling interval. Fuel prices are not second-by-second telemetry, and MIMIT is a public service. The integration defaults to `60 minutes` and enforces a minimum of `30 minutes`.

## Changelog And Updates

This repository includes `CHANGELOG.md`. When publishing updates:

1. Update `custom_components/fuel_prices_italy/manifest.json` with the new version.
2. Add a new section to `CHANGELOG.md`.
3. Create a GitHub release with the same tag, for example `v0.1.1`.
4. Put the changelog entry in the GitHub release notes.

HACS and Home Assistant update views can then show release information and changelog content from the repository release.

## Known Limitations

- The integration currently uses manual coordinates only.
- It does not yet provide a browser GPS button.
- MIMIT does not provide a formal public developer contract for this endpoint; schema changes are possible.
- Station addresses are often returned as `null` by the endpoint.
